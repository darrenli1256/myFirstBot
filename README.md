# myFirstBot
Darren's first chatbot (Omnichat tech test)

### Instruction
1. fill the blank in .env file including
	- LINE_CHANNEL_ACCESS_TOKEN
	- LINE_CHANNEL_SECRET
	- PORT
2. Install ngrok and run "ngrok http (PORT)" on terminal
3. Testing 
	- WebHook API : (domain generated from ngrok)/webhook 
	- HTTP method: POST
